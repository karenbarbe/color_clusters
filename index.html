Color clusters

How it works

Color Clusters help you visualize an enhanced version of colors obtained from the Image Color Summarizer API by adjusting their saturation and lightness.

While the API accurately captures hue values, it returns averaged colors that can appear muddy, especially when the palette is intended for visual projects.

The Color Clusters script aim to bring out the vibrant and clean colors that our eyes perceive in the image by boosting saturation and adjusting lightness levels as shown in the following chart.

Color corrections

Hue

  • Original hue value

Saturation

  • Light colors (L ≥ 70): Increase by ~25%
  • Mid-range colors (40 ≤ L < 70): Increase by ~20%
  • Dark colors (L < 40): Increase by ~13%

Lightness

  • Light colors: Increase by 5-6 points
  • Mid-range colors: Increase by 13-14 points
  • Dark colors: Increase by 10 points

How to use this script

  • Clone the repository
  • Follow README file to create virtual environment and install packages
  • Update image_name and/or url to match the image location in main.py
  • Update imageName variable in script.js
  • Update num variable in script.js to adjust clusters quantity
  • Run main.py and open index.html
